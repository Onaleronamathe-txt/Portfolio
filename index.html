#Projects

Study Hub
Full Stack Web Application
● LIVE DEMO BELOW Public In Progress

A full-stack student Q&A and academic management platform. Students can post questions, vote on answers, track grades with GPA calculation, and manage personal study notes — all in one place. Built with a Java Spring Boot REST API backend, MongoDB database, React frontend, and containerised with Docker Compose for production deployment.

Key engineering highlights: RESTful API design with 10+ endpoints, JWT authentication, MongoDB document modelling with Spring Data, React component architecture, full Docker Compose deployment — the kind of system design you'd build on a real team.
